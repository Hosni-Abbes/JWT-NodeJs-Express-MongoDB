Welcome To This Demo Website.

It Is a Website To Deal With JWT (JsonWebToken).

I use Nodejs, Express and MongoDB For the Backend.

You Can Use user Data Bellow to Login as a User.

You Can Use admin Data Bellow to Login as Admin.

AdminData => Username: admin123 ## Password: admin123 User.Data => Username: user123 ## Password: user123

Or You Can Create an Account.


When you login as User you can delete only your data.

When you login as Admin you can delete all other data.

This actions provided by using Json Web Tokens.
